Our-sphere

a portable 3D data-map viewer for iOS, Webkit browsers

Download this project as a .zip file Download this project as a tar.gz file

Author Taiyo Fujii

Team: takano21 , hagino3000

API

Standard Earth Map

[service URL]

ex)http://taiyolab.com/ourspheredev/

Importing Image by URL

[service URL] + /?map=[escaped Image URL]

ex)http://taiyolab.com/ourspheredev/?map=http%3A//rapidfire.sci.gsfc.nasa.gov/imagery/firemaps/firemap.2011271-2011280.2048x1024.jpg

Showing Satellite(s)

[service URL] + /?satellite=true

ex)http://taiyolab.com/ourspheredev/?satellite=true

Usage as Widget

<iframe class="our sphere"
        src="http://taiyolab.com/ourspheredev/index.html"
        width="200" height="200">
</iframe>
